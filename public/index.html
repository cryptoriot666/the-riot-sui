THE PROBLEM

Agents Are Stateless. Memory Is Fragile.

Today's AI agents complete tasks in isolation, lose context across sessions, and can't share knowledge across tools or teams.

Session Amnesia

Every conversation starts from zero. Agents forget preferences, history, and personality the moment the tab closes.

Siloed Memory

Memory is locked to single apps or models. Agents can't share context across workflows, teams, or platforms.

Dead NFTs

Static JPEGs in wallets with no ongoing utility. No interaction, no evolution, no reason to hold beyond speculation.

MEMORY PIPELINE

How It Works

From mint to memory persistence. Every step is verifiable on Walrus.

01

Mint + Bind

Mint J4 (or any agent). Smart contract assigns unique Agent ID and generates MemWal Delegate Key.

agent_id: J4-0x9f2a
delegate_key: 0x7a3f...
memwal_space: initialized
02

Chat & Remember

Talk via Telegram. Every message is written to MemWal (Walrus Memory), not local session storage.

> user: "I like neon cats"
> J4: [writes memory]
> walrus_blob: confirmed
03

Delegate & Coordinate

J4 reads shared MemWal context and delegates to specialist agents (Artist, Lore, Strategy) via Sui Stack Messaging.

> J4 -> J10: "need art"
> context: shared_memwal
> J10 reads: "neon cats"
04

Store & Evolve

Generated artifacts stored on Walrus. Blob ID recorded in Move contract. NFT becomes a living document.

> artifact: cyberpunk.png
> walrus_blob: 0x3d8e...
> nft_metadata: updated

ARTIFACT-DRIVEN WORKFLOW

Visual Artifacts

Memory Logs
